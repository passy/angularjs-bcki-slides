Dependency Injection

  • Drei Möglichkeiten an eine Abhängigkeit zu gelangen:
    1. Erzeugen, etwa über new-Operator
    2. Auflösen über eine globale Variable
    3. Übergeben wo es benötigt wird
  • DI ist die dritte Option

Dependency Injection

Beispiel: HTTP

function FetchController($scope, $http, $templateCache) {
    $scope.fetch = function () {
        $http({method: 'GET', url: $scope.url, cache: $templateCache}).
            success(function (data, status) {
                $scope.status = status;
                $scope.data = data;
            }).error(function (data, status) {
                $scope.status = status;
                $scope.data = "Oh noes!";
            });
    };
}
